## Description
Habit Tracker is a client-side web application designed to help users build and maintain daily habits. It provides an easy and interactive way to track progress and stay motivated.

## Features
- **Responsive Design:** Works perfectly on desktops, tablets, and mobile devices.
- **Interactive UI:** Add, edit, and delete habits with smooth animations.
- **Progress Tracking:** Visualize your habit completion with charts or counters.
- **User-Friendly Navigation:** Simple interface for quick access to your habits.
- **Modern Design:** Built using HTML, CSS, and JavaScript for a clean, interactive experience.

## Technologies Used
- HTML
- Tailwind CSS
